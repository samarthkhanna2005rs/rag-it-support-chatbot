IT Support Knowledge Base

Password Reset Procedure:
1. Open company password portal.
2. Click "Forgot Password".
3. Verify identity using employee ID and OTP.
4. Create a new password with at least 12 characters.

VPN Troubleshooting:
1. Verify internet connection.
2. Restart VPN client.
3. Confirm VPN credentials.
4. Reconnect to corporate gateway.
5. Contact IT if issue persists.

Software Installation Policy:
Employees must request approval before installing external software.

Printer Troubleshooting:
1. Check printer power.
2. Verify network connectivity.
3. Restart printer service.
4. Reinstall printer drivers if required.
